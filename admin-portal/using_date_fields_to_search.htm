Time Range Filters

You can use filters for a specific time range when you create search queries or generate reports. The available options for time range filters are:

  • During (either in the past or the future)
  • More than

Options for "During" in the Past

Time range

Explanation

Last 24 Hours

The last completed day.

Last 7 Days

The last completed seven days plus any elapsed time today. For example, if the current time is Friday at noon, the range for the last seven days ends at Friday (today) noon and starts at a week ago from yesterday (Thursday).

Previous Week

The last completed Sunday to Saturday calendar week. For example, if today is Wednesday, the range ends on the first previous Saturday and starts on the Sunday before the previous Saturday.

Last 30 Days

The last completed 30 days plus any elapsed time today. For example, if today is Tuesday at 10:00 a.m., the range for the last 30 days ends at Tuesday (10:00 a.m.) and starts 30 days ago from yesterday (Monday).

Previous Month

The last completed calendar month. For example, if this month is July, the previous June is the range for the report or search.

Last 2 Months

The last two completed calendar months. For example, if this month is November, the range for the report or search is September and October.

Last 3 Months

The last three completed calendar months. For example, if this month is May, the range for the report or search is February, March, and April.

Previous Quarter

The last completed calendar quarter where quarters start and end with the calendar year (January to March, April to June, July to September, October to December).

Last Year

The last completed calendar year (January to December).

Options for "During" in the Future

Time range

Explanation

Next Week

The next complete Sunday to Saturday calendar week.

Next Month

The next complete calendar month. For example, if this is August, the report is generated for September.

Next 2 Months

The next two complete calendar months. For example, if this is April, the report is generated for May and June.

Next 3 Months

The next three complete calendar months. For example, if this is October, the report is generated for November, December and January.

Next Quarter

The next complete calendar quarter where quarters start and end with the calendar year (January to March, April to June, July to September, October to December).

Next Year

The next completed calendar year (January to December).

Options for "More Than"

Time range

Explanation

7 Days Ago

Anything occurring on the day that was seven days ago and before today.

30 Days Ago

Anything occurring on the day that was 30 days ago and before today.

90 Days Ago

Anything occurring on the day that was 90 days ago and before today. This selection isn't available for Last Logon Time.
