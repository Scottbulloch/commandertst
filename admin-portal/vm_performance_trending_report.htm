VM Performance Trending Report

For a selected VM, use the VM Performance Trending Report to:

  • analyze the growth or decline of that VM's resource consumption in the past
  • analyze the projected growth or decline of that VM's resource consumption in the future

See VM performance metrics for details on the metrics included in the report.

VM performance data is automatically updated each night. To make sure your report contains the latest performance data, you can update VM performance before running the report.

Generating the VM Performance Trending Report

Access:

Reports > Capacity & Performance > VM Performance Trending

Available to:

All Access Rights Levels

To generate the report:

  1. To specify the date range for the report, select one of the following:
    • By Period: Allows you to specify a date range by period. If you select this option, in the During field, select a period of time.
    • By Dates: Allows you to specify a date range using a date picker. If you select this option, use the date picker to select a From Date and a To Date.

    Make sure that you select a date before the last purge date. Otherwise, the report may display incorrect values.

  2. From the Calculate Using drown-down, select one of the following calculation methods:
    • Average values: Calculates VM performance for the selected date range using average values for resource usage.
    • Peak values: Calculates VM performance for the selected date range using peak values for maximum resource usage.
  3. In the VM Location field, click to select a VM, and click OK.
  4. In the Interval field, select the report interval.
  5. In the Trend field, select one of the following:
    • Yes: Shows the Trend column in the Performance Trend table.

      In the Trend column, a positive number indicates an upward trend in resource consumption, a negative number indicates a downward trend, and a zero value means resource consumption isn't projected to change.

    • No: Hides the Trend column in the Performance Trend table.
  6. Click Generate.

If you generated the report by navigating to a VM and running the Generate Performance Trending Report command, go to Reports > Generated Reports, select the report in the list, and click View.
